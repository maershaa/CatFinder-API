# Проект: Интерактивный веб-сайт с котами, погодой и рандомными актёрами

## О проекте

Этот проект представляет собой интерактивный веб-сайт, где можно:

- **Выбирать породу кошки** из списка, получаемого через API, и после выбора
  видеть карточку с **информацией о породе и фотографией**.
- **Получать текущие данные о погоде** на основе вашей локации.
- **Просматривать случайного актёра** из базы фильмов через виджет, который
  запускает запрос при клике.

Проект использует нативный Fetch API с async/await для выполнения асинхронных
запросов к внешним API и динамически обновляет интерфейс без перезагрузки
страницы.

---

## Используемые технологии

- **HTML / CSS / JavaScript** — структура, стили и логика.
- **[UX Select](https://ux-select.com/#introduction-description)** — кастомный
  селект для выбора породы кошки.
- **[CSS Loaders](https://cssloaders.github.io/)** — анимация загрузки данных.
- **[Toastify](https://github.com/apvarun/toastify-js)** — уведомления об
  ошибках и успехе.

---

## Используемые API

- **[The Cat API](https://thecatapi.com/)** — список пород кошек, изображения и
  информация о них.
- **[WeatherAPI](https://www.weatherapi.com/)** — данные о текущей погоде.
- **[TMDB (The Movie Database)](https://www.themoviedb.org/)** — данные о
  фильмах и случайных актёрах.
